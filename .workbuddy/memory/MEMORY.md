# 项目记忆 - 集团模拟经营文字游戏

## 项目概述
手机可玩的文字类集团模拟经营游戏。主界面分金融/实业两类按钮，点击跳转子页面，底部固定总览栏。

## 技术栈
- H5 单页应用（HTML/CSS/JS）
- PWA 配置（可加桌面、离线可玩）
- Capacitor 打包成 Android APK（后续）
- localStorage 存档
- 部署：CloudStudio 云端部署，手机扫码体验

## 设计规范
- 视觉风格：简洁卡片风（白底卡片、0.5px边框、圆角、系统字体）
- 移动端优先：竖屏布局、按钮触摸区≥44px、大字号宽松行距
- 中国股市惯例：涨红跌绿、¥货币符号

## 板块结构
- 金融：存款、股票
- 实业：工厂、农业、钢铁、矿业、地产
- 底部总览：集团概览、存款、产业总览

## 时间设定
- 起始日期：2018年1月1日（周一）
- 推进方式：玩家点击驱动，非实时
- 加速档位：×1（看结算明细）/ ×7（遇事件暂停）/ ×30（月末汇总）
- 事件系统可结合真实经济史增强代入感

## 开发顺序
1. 骨架 + 主界面布局
2. 各板块子页面
3. 每日结算逻辑 + 事件系统
4. localStorage 存档
5. PWA 配置 + 云端部署
6. Capacitor 打包 APK
