# 2026-07-11 工作记录

## 暗色模式全面修复

### CSS 暗色覆盖补全（style.css）
- `.tab.active`：新增 `[data-theme="dark"]` 覆盖，使用 `var(--text-primary)` 背景 + `var(--bg-card)` 文字
- `.time-bar-fill`：新增暗色渐变（`var(--info)` → `#6bb5ff`）
- `.modal-mask`：暗色加深至 `rgba(0,0,0,0.6)`
- `.np-slider` 滑块拇指：暗色使用 `var(--text-primary)`

### index.html

### Canvas 图表深色适配（charts.js）

- `sparkline()` 和 `lineChart()` 默认颜色改为 `_themeColors()` 动态读取
- 修复零线标签 "楼0" 错别字 → "0"

### 硬编码颜色清理
- `warehouse.js`：移除 `background:rgba(186,117,23,0.1)`（已有 `border-left: var(--warning)` 足够）
- `industry.js`：同上
- `industryDetail.js`：修复 `.btn` 重复 `color:var(--info);color:#fff;` → 合并为 `color:#fff`

## 选择难度后导航失效修复
- 问题：`startGame()` 从 onclick 事件内调用 `Router.goRoot('overview')` 替换 DOM，被残留的事件上下文吞掉
- 修复：`Router.goRoot('overview')` 改为 `setTimeout(() => Router.goRoot('overview'), 0)`，推入事件队列末尾，确保 DOM 稳定后执行导航
